📸 Photo Migration Tool

Migrates photos from old Firebase Storage URLs into cmk63project Storage, then updates the database URLs.

Step 1 — Sign In to cmk63project

Step 2 — Scan Database for Photos

⚠ This tool will:
1. Scan every person in your genealogy database for photo URLs
2. Download each photo from the old Firebase Storage URL
3. Re-upload it to cmk63project Firebase Storage
4. Update the person's record in the database with the new URL

Base64 photos already in the database will be uploaded to Storage and replaced with URLs too.

Run a dry run first to see what will be migrated before making any changes.
